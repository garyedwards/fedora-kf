fedora-kf a minimal fedora remix

The idea

The idea is to create a kickstart file and set of configs that can be used to install Fedora GNU/Linux, how I like it out of the box. I may create a Live CD in the future.

Usage

You can point your Fedora install directly at the Kickstart file (this will not work at the moment as I have not moved all of the files across): 

http://kf.edwards.org.je/fedora-kf.ks

You can also clone the project with Git by running: 

$ git clone git://github.com/garyedwards/fedora-kf

get the source code on GitHub : garyedwards/fedora-kf
